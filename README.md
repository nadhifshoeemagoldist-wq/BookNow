## 📌 Deskripsi

**BookNow** adalah aplikasi web berbasis PHP dan MySQL yang digunakan untuk melakukan pemesanan (booking) layanan secara online dengan mudah dan cepat.

Aplikasi ini memungkinkan pengguna untuk:

* Memilih layanan
* Menentukan tanggal dan waktu booking
* Melihat status booking
* Mengakses riwayat pemesanan

---

## ✨ Fitur Utama

* 🏢 Daftar layanan (Futsal, Barber, Konsultasi, dll)
* 📅 Sistem booking tanggal & jam
* 🔄 Status booking (Pending, Disetujui, Selesai)
* 📜 Riwayat booking pengguna
* 🔐 Login admin
* 🛠️ Admin panel untuk mengelola booking
* 🚫 Validasi anti double booking

## ⚙️ Cara Menjalankan

1. Clone repository:

   ```bash
   git clone https://github.com/USERNAME/BookNow.git
   ```

2. Pindahkan ke folder:

   ```
   htdocs/
   ```

3. Import database:

   * Buka phpMyAdmin
   * Buat database: `booknow`
   * Import file SQL

4. Jalankan di browser:

   ```
   http://localhost/BookNow
   ```
